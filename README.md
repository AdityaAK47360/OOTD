# OOTD

## Setup

1. `npm install`
2. `npm run dev`
3. Open the provided local URL (default `http://localhost:5173`)

## Build

- `npm run build`
- `npm run preview`

## Project structure

- `index.html` - entry template
- `src/main.jsx` - app bootstrap
- `src/App.jsx` - main React component
- `src/data.js` - dataset for weather/mood outfits
